== Description ==

Crossword - word puzzle game that takes the form of a square or a rectangular grid of squares. The game's goal is to fill the white squares with letters, forming words or phrases, by solving clues, which lead to the answers.

== Installation ==

1. Activate Plugin
2. Add new Wordsearch
3. Copy Wordsearch shortcode and paste to any page or post
4. Set Default or individual setting for each game.
